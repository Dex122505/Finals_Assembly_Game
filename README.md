# Finals_Assembly_Game

Branches: 

1.  MAIN - This is where the final game will be stored.
2.  PROGRAMS - This branch is for programming task related to game mechanics, logic, and other features.
3.  SOUND-DESIGNER - This branch is for sound-related tasks, including background music and other sound effects.
4.  UI-UX-Design - This branch is for UI/UX design task. You can upload the PDF here for wireframes and other design.
5.  Visual Effects - This branch is for tasks related to animations and visual effects.

NOTE: 

1. COMMIT FREQUENTLY
   - Commit your changes regulart to save your progress and make it easier to track changes.
  
2. COMMENTING CODE FOR READABILITY
   - Write comments in your code to explain what is meaning of that line or logic of the program.
  
3. KEEP CODE ORGANIZED
   - Make the layout of the code consistent.
   - If code is not needed, delete it instead of commenting it out to keep the file clean.
